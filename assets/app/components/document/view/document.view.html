- How to build and run:

  • Install Docker: https://www.docker.com/community-edition#/download
  • Run MySQL docker container: docker run -d --name elevator_db -e MYSQL_PASS=bodyguy -e ON_CREATE_DB="elevator" tutum/mysql
  • Run My Elevator App docker container: docker run --name elevator_app --link elevator_db -e NODE_ENV=development -e ENV_RUN_CMD='node app.js' -p 1337:1337 ducbaovn/elevator
